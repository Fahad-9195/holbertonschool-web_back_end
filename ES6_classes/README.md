# ES6 Classes

## Description

This project covers Object-Oriented Programming using ES6 classes in JavaScript. Each task demonstrates a specific class concept through practical examples.

## Requirements

- NodeJS 12.11.x
- npm 6.11.3
- Jest, Babel, and ESLint

## Tasks

| File | Concept |
|------|---------|
| `0-classroom.js` | Basic class definition |
| `1-make_classrooms.js` | Creating class instances |
| `2-hbtn_course.js` | Getters and Setters |
| `3-currency.js` | Class methods |
| `4-pricing.js` | Static methods |
| `5-building.js` | Abstract class pattern |
| `6-sky_high.js` | Class inheritance |
| `7-airport.js` | Symbol.toStringTag |
| `8-hbtn_class.js` | Symbol.toPrimitive |
| `9-hoisting.js` | Class hoisting |
| `10-car.js` | cloneCar method |
